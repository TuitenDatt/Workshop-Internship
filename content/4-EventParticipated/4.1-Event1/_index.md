# Summary Report: "FCAJ Community Day"

## Event Information

- **Event Name:** FCAJ Community Day
- **Date:** June 6, 2026
- **Time:** 09:00 AM – 12:00 PM
- **Location:** Bitexco Financial Tower, Ho Chi Minh City
- **Role:** Attendee

# Event Objectives

FCAJ Community Day was organized to share knowledge and practical experience in the fields of Cloud Computing, Cybersecurity, DevOps, software development, and teamwork skills. Through presentations by various speakers working in the industry, the event helped attendees gain exposure to modern technologies and system models currently applied in enterprises, while broadening career directions in Information Technology.

The event aimed to achieve the following goals:

- Introduce modern technologies and solutions in Cloud and Security.
- Present practical system architectures in software development.
- Share career transition experiences in the IT industry.
- Guide the application of DevOps tools and containerization in system deployment.
- Enhance teamwork skills in technology projects.

---

# Session 1: AWS WAF & Machine Learning NIDS

The presentation by Gia Dai Le Hoang focused on combining AWS WAF with Machine Learning to improve cybersecurity threat detection and prevention.

### Key Topics

- Analyzing limitations of traditional AWS WAF when relying solely on rule-based detection, making it difficult to detect new or Zero-day attacks.
- Introducing the Network Intrusion Detection System (NIDS) utilizing Machine Learning.
- Using the CSE-CIC-IDS2018 dataset to train models like LightGBM to detect anomalous behaviors (Botnet, DoS, Brute Force).
- Real-time collection, processing, and monitoring architecture deployed on AWS consisting of EC2, S3, Kinesis, Lambda, and CloudWatch.

### Key Takeaways

- Machine Learning enhances attack detection capabilities compared to traditional rule-based methods.
- AWS architecture provides great support for real-time cybersecurity monitoring systems.
- Combining multiple AWS services helps expand data processing and analysis capabilities.

---

# Session 2: Docker – Containerization Technology

The presentation by Bao Huynh introduced Docker and the Containerization model in modern software development.

### Key Topics

- The concept of Containerization: packaging an application with all its dependencies to ensure consistent operation across any environment.
- Comparison of Containerization and Virtualization, where containers are lighter, faster, and more resource-efficient.
- Introduction to Docker components: Docker Image, Dockerfile, and Docker Engine.
- Basic Docker commands for container management.
- Application of Docker in CI/CD, Microservices, and legacy system modernization.

### Key Takeaways

- Docker helps standardize application development and deployment environments.
- Containerization is a crucial foundation of Cloud Native architecture.
- Docker accelerates DevOps workflows and software deployment.

---

# Session 3: From IT Helpdesk to Senior Sysadmin

The presentation by Trung Vinh Tran shared the career development journey from IT Helpdesk to Sysadmin and orientation towards Cloud/DevOps.

### Key Topics

- Experience working in Helpdesk positions: troubleshooting and supporting users under high pressure.
- Self-learning foundational platforms like Linux, Networking, and building lab environments for practice.
- Developing system thinking: automating repetitive tasks and building runbooks.
- A critical principle in operations: "Never test on production".
- Transitioning to Cloud/DevOps with tools like Terraform (Infrastructure as Code) and Docker.

### Key Takeaways

- Foundational skills are essential for IT career development.
- Self-learning and hands-on practice are keys to career transition.
- DevOps and automation are inevitable trends in the technology industry.

---

# Session 4: Connecting Godot Client with AWS WebSockets

The presentation by Quoc Bao Nguyen focused on building multiplayer game systems on Cloud platforms.

### Key Topics

- Comparison of networking models: UDP/ENet, HTTP Polling, and WebSocket.
- Reasons for choosing WebSocket for Lobby/Turn-based game systems.
- AWS architecture consisting of API Gateway (WebSocket), AWS Lambda, and DynamoDB.
- Integrating WebSocket in Godot 4 using WebSocketPeer and JSON messaging.
- Real-world challenges such as DynamoDB Scan costs and handling stale connections.

### Key Takeaways

- WebSocket is suitable for real-time applications like multiplayer games.
- Serverless architecture on AWS makes system scaling easy.
- System design needs to optimize both performance and cost.

---

# Session 5: The Art of Effective Teamwork

The presentation by Huy Phuoc Truong focused on teamwork skills in technology projects.

### Key Topics

- Four principles of effective teamwork:
    - Clear and aligned goals.
    - Delegation of the right tasks to the right people.
    - Open communication and active listening.
    - Personal responsibility in work.
- Supporting tools: Trello, ClickUp, Google Workspace, Slack, and Discord.

### Key Takeaways

- Effective teamwork depends on communication and personal responsibility.
- Management tools help optimize project performance.
- Good collaboration enhances product quality.
